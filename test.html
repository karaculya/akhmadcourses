Тест по теме Тестирование ПО

Данный тест предназначен для проверки знаний по основам тестирования программного обеспечения. Тест состоит из 5 вопросов, каждый из которых имеет один правильный ответ. Время на прохождение теста не ограничено, но рекомендуется не тратить на каждый вопрос более 2-3 минут.

Оценка результатов:

За каждый правильный ответ начисляется 1 балл. В конце теста выдается итоговая оценка в процентах.

отлично 90%
хорошо 70%
удовлетворительно 50%
неудовлетворительно < 50%
